### Installation
Install Prefect:
```bash
pip install dbt-bigquery
```
Check version:
```bash
dbt --version
```
Create a directory ie 'dbt':
```bash
mkdir dbt
```
Move to directory.  
Initialize dbt:
```bash
dbt init
```
This will prompt you to name your dbt project and ask for details based on the database you choose.
